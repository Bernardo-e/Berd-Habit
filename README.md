# 🌱 Berd Habit

### Build Consistency. One Day at a Time.

Berd Habit is a modern habit tracking web application designed to help users build positive routines, stay consistent, and visualize progress through an engaging plant growth system. Every completed habit contributes to growth, transforming discipline into a rewarding visual journey.

---

## ✨ Features

### 🌱 Habit Management

* Create habits
* Edit habits
* Delete habits
* Organize habits with categories

### 🔥 Streak Tracking

* Current streak tracking
* Longest streak tracking
* Daily completion system
* Consistency monitoring

### 🌳 Plant Evolution System

Watch your virtual plant grow as your discipline grows.

Growth Stages:

* 🌱 Seed
* 🌿 Sprout
* ☘️ Small Plant
* 🌷 Blooming Plant
* 🌳 Young Tree
* 🌲 Flourishing Tree

### 📈 Progress Analytics

* Daily progress tracking
* Weekly insights
* Monthly consistency overview
* Completion percentages

### 🎨 User Experience

* Responsive design
* Dark mode support
* Smooth animations
* Beautiful micro-interactions
* Motivational growth journey

### 💾 Data Persistence

* Local Storage support
* Progress saved automatically
* Habit history maintained across sessions

## 🎯 Philosophy

Berd Habit is built around a simple idea:

> Small actions repeated consistently create meaningful growth.

The plant evolution system serves as a visual reminder that discipline compounds over time, turning daily habits into long-term progress.
